# Physics Educational Platform

## Demo:

This platform is a website for university students to make them learn easier 4 categories of physics with 4 virtual experiments:
- Mechanics: Double Pendulum.
- Waves & EM: Wave Interference.
- Quantum Physics: Particle in a Box.
- Thermodynamics: Kinetic Theory of Gases.

Every experiment has the principal screen with the animation and 3 menus:
- Control: to give parameters for adjusting the experiment.
- Theory: Learn the technical part of every experiment with formulas.
- Quiz: Make sure you understood the theory.

For the demo I implemented the 4 basic domains of physics paired with one experiment for every domain.

## 🛠️ Technical Information:
Website made mainly with JavaScript(functionality), html(web page), css(design) and KaTeX(for the Theory menu).
Hosted online using GitHub Pages.

##  📖 Tutorial:
Go to the demo link and in the left part you will see the 4 domains and choose 1 and in the middle you will see the experiment in the inferior part some statistics and in the right part the 3 menus.
